# Getting Started

:::danger

THIS ENTIRE PLATFORM IS CURRENTLY A WORK-IN-PROGRESS (OR WIP) &
IS NOT YET READY FOR PUBLIC USE.
 
WITH REGARD TO PLUGINS, NO CONCERN HAS BEEN GIVEN TO SECURITY.

USE AT YOUR OWN RISK.

:::

[__*VRKit Platform*__](https://vrkit-platform.com) is designed as a fully customizable platform 
for creating & rendering via a standard monitor(s) and/or [_OpenXR_](https://www.khronos.org/OpenXR) HMD Devices.

:::info 
The [_OpenXR_](https://www.khronos.org/OpenXR) Layer implementation & much of the IPC implementation 
is derived from the work of [Fred Emmott](mailto:fred@fredemmott.com), specifically [Open Kneeboard](https://openkneeboard.com/), 
the source is available on [Github](https://github.com/OpenKneeboard/OpenKneeboard)
:::

## Dependencies

The _*VRKit*_ Installer will also install all dependencies required:

- DirectX
- DotNET Desktop Runtime
- Windows App Runtime

## Download & Install

the warning window, you can press `More Info` & then `Run Anyway`.

By doing this you assume all risk related to using this software.

## Launch _*VRKit*_

The first launch of _*VRKit*_ should generate your a default dashboard consisting
of a `trackmap` & `clock` overlay.

You should be looking at the dashboards list, with your newly generated dashboard set as the default.
